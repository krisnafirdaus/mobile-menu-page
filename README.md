# Mobile Menu Page

A Flutter app that implements a responsive menu page with selectable menu details.

## Requirements

- Flutter SDK
- iOS Simulator or Android Emulator

## Run

```bash
flutter pub get
flutter run
```

If Flutter is not available in `PATH`, run it from the local SDK:

```bash
/Users/krisnafirdaus/flutter/bin/flutter run
```

## Check

```bash
flutter analyze
flutter test
```
